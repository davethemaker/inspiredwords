# inspired words
## June 3, 2017
Description:
  express framework with CRUD functionality to mongoLab db,
  EJS template engine, vanilla JS vs jQuery for eventListeners

**Installation:**
1. git clone repo
2. [cli] $ npm install
3. [cli] $ node server.js (or, $ npm run start)
4. [browser] localhost://8080
